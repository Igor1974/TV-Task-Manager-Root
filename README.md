TV Task Manager — Умный диспетчер задач для TCL и Android TV

Версия: 2.0.1 (Stable) Разработчик: ~Igor~(4PDA) Системные требования: Android 9+, ROOT (обязательно)
Единственный в своем роде диспетчер задач, созданный специально для телевизоров (оптимизировано под TCL).
Переход на новый движок программы Apex Engine, кастомный движок, созданный для прямого взаимодействия с низкоуровневыми ресурсами Android TV.
В отличие от обычных Task Killer-ов и ADB-утилит, программа работает на системном уровне, обеспечивая глубокую очистку и автоматизацию, которая раньше была доступна только в кастомных прошивках.

Основной функционал:
Перехват кнопки Netflix: Вдохнул жизнь в бесполезную кнопку на пульте.
Теперь одно нажатие на Netflix мгновенно вызывает диспетчер задач поверх любого приложения. (Если кнопки нет — работает долгое нажатие HOME).
Интеллектуальная защита: Система «Safety First» блокирует возможность случайного закрытия критических процессов TCL. Телевизор не зависнет и не уйдет в ребут.
Оптимизированные алгоритмы опроса ресурсов — программа «невидимка» для системы, которая не отнимает ресурсы у воспроизведения видео.
Настоящий Автозапуск (Wake-on-Screen): Программа мониторит выход ТВ из режима ожидания (Standby) и автоматически запускает выбранные вами приложения.
Delta-мониторинг CPU: Показывает реальную, а не среднюю нагрузку на ядра процессора в режиме реального времени.
Глубокая очистка: Удаление скрытых логов TCL (tcl_log) и системных дампов, которые могут «съедать» до 2 Гб памяти.
Глубокая заморозка: Отключайте системный «мусор», который нельзя удалить, чтобы освободить оперативную память.

Инструкция по управлению:
Вызов меню: Кнопка Netflix или удержание HOME (2-3 сек).
Действия в списке: Удержание OK на приложении открывает меню (Автозапуск / Белый список / Заморозка и другие).
Монитор ресурсов: Нажатие кнопки «МОНИТОР» включает оверлей, который показывает: Температура CPU, Свободная RAM, Скорость сети.
Долгое нажатие кнопки "ВХОДЫ" на пульте, позволяет выбрать позицию оверлея.
Белый список: Приложения игнорируются при нажатии «Закрыть всё».

Важное примечание:
При первом запуске обязательно предоставьте права Superuser (Magisk) и разрешите наложение поверх других окон.
Для работы перехвата кнопок программа автоматически попросит включить службу спец. возможностей (Accessibility Service).
